# Rash 🦀 - reasonable again shell

A bash 🐚 alternative with reasonable syntax, written in Rust.

# Basic syntax 📜
```rash
  var x = 5
  print("hello world")
  ~ ls
```
# Types 📦
- string ("hello world" || 'hello world')
- int
- float
- bool
